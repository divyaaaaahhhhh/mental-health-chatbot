# Mental Health AI Chatbot

This is a simple AI-based chatbot built using Flask, HTML, CSS, and JavaScript.  
It provides basic conversational support and a friendly chat interface.

## Features
- Simple chatbot UI
- Real-time user interaction
- Flask backend integration
- Lightweight web app

## Tech Used
- Python (Flask)
- HTML
- CSS
- JavaScript

## How to Run
1. Install requirements:
   pip install -r requirement.txt

2. Run the app:
   python app.py

3. Open browser:
   http://127.0.0.1:5000
